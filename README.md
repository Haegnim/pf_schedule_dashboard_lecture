## 주요기능

- 일정 쓰기/수정/삭제 구현
- php을 이용해 DB 사용
- 일정을 json으로 저장, 각각의 대시보드에 데이터를 불러와 반영
- 각 프로젝트 일정 필터링 보기
- 웹 시작시 비밀번호 입력 화면으로 넘어가도록 구현
